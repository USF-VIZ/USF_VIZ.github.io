Data Set

The first data set has three files with the results of applying two different machine learning algorithms to the same dataset. The files results-glmnet-cross-table.csv and results-pgboost-cross-table.csv show two error metrics rmse and mederr which are the result of applying 100 crossvalidations to the same dataset. Here each row is an iteration of the algorithm.

The file results-gradient-boosting.csv shows detailed results for one run of gradient boosting. Here each row is one data point.
